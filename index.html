How It Works

Powerful Results

From registration to analytics — the entire workflow in four seamless steps.

Step 01
Register Your Center
Admin onboards your Aadhaar center with a unique ID, geo-location tag, and operator assignment — all verified through the UCL owner portal before activation goes live.
Step 02

Create operator profiles with granular permission controls. Each agent gets a role-specific view — no data leakage, no unauthorized access, fully auditable at every level.
Step 03
Submit & Track PAC Entries
Operators submit PAC records via the secure entry form. Every submission is timestamped, validated instantly, and immediately reflected in the live dashboard with full status tracking.

Analyze & Generate Reports
Admins access real-time analytics, monthly performance summaries, and exportable reports — all powered by Firestore with sub-second query response times on any device.

FAQ

Frequently
Asked Questions

Everything you need to know about PAC Management — answered clearly.

How does real-time syncing work across multiple centers?
PAC Management uses Google Firestore's native real-time listeners. The moment any operator submits a PAC entry or updates a record, that change propagates to every connected dashboard in under 500ms — no polling, no manual refresh, no delays.
What happens if the internet connection drops during a submission?
Firestore's offline persistence caches pending writes locally. When the connection is restored, all queued submissions sync automatically in order. No data is lost, and operators can keep working without interruption even in low-connectivity environments.
How are operator roles and permissions managed?
Admin users can define granular role-based access control (RBAC) for each operator. Roles include View-Only, Data Entry, Center Manager, and Full Admin. Firebase Security Rules enforce these permissions at the database level, so even direct API access is protected.
How often does the automatic backup run?
Backups are triggered every 30 seconds for active sessions, and a full daily snapshot is stored in Google Cloud Storage with 30-day retention. You can also manually trigger a backup at any time from the admin dashboard.
Can I export PAC data to Excel or PDF?
Yes. The Reports module lets you export any filtered dataset to CSV, Excel (XLSX), or a formatted PDF report. You can filter by date range, center, operator, or status before exporting. Scheduled email reports are also available for admin accounts.
Is the system mobile-friendly for field operators?
Fully. PAC Management is a Progressive Web App (PWA) with a responsive layout optimized for phones and tablets. Operators can submit PAC entries, check statuses, and receive notifications directly from their mobile browser — no app install required.
